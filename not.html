NOT Gate

The NOT Gate is a digital component that performs a bitwise inverse between specified bits. This basic level gate accepts one input value A and the inverse of A will be output on value Y.

Truth Table

The Truth Table for the NOT Gate is quite simple. If a logic level A = "0" is inserted, then a "1" will be produced to Y. If the logic level input of A is equal to "1", then Y will output a "0".
