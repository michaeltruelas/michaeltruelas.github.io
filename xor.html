XOR Gate

The XOR Gate is a digital component that performs bitwise XOR operations between specified bits. This basic level gate accepts two input values A and B and performs the logical XOR(⊕) operation which will be output on value Y.

Truth Table

The XOR Gate is often used as a comparator because the output of Y is dependent on the relationship between the inputs. Specifically, the inputs A and B must be different from each other to produce a "1" output in Y.
